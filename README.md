# Codepath TIP103: Advanced Technical Interview Prep

Advanced Technical Interview Prep - Summer 2026

This repository contains solutions and practice problems for the Codepath TIP103 course.

## How to Run

Run any problem file individually using Python:

```bash
python problem1.py
python problem2.py
python problem3.py
```

Or with explicit path:
```bash
python "Problem 1: Transpose Matrix.py"
```
